¡Bienvenido a la documentación de Hermes BookSearch!
====================================================
**Hermes BS** es una aplicación de uso sencilla e intuitiva que le permitirá administrar una biblioteca doméstica, registrando así sus libros en stock con sus datos. La aplicación se haya construída integramente sobre **python**, haciendo uso de una interfaz desarrollada en **tkinter** y una ligera base de datos **SQLite** para guardar la información.
